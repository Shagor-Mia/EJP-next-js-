# **App Name**: EventFlow

## Core Features:

- Landing Page: Display a landing page with a Navbar, Hero section, relevant content sections, and a Footer. Navbar updates after login with logged-in user info and options to add/manage events.
- Authentication: Implement user authentication using NextAuth.js, including social login (Google) and credentials form for login/register pages.
- Event Listing: Display a list of events with a search bar and optional category filter. Each event card includes an image, title, short description, price/meta, and a details button.
- Event Details: Show detailed information for each event on a dedicated page, including a large image, event title, full description, and meta info.
- Protected Add Event Page: Enable logged-in users to add new events via a form with fields for title, description, price, date, priority, and an optional image URL.
- Protected Manage Event Page: Allow logged-in users to manage existing events, with options to view and delete events, presented in a table/grid layout.

## Style Guidelines:

- Primary color: Deep violet (#673AB7), chosen to convey creativity, energy, and a sense of community engagement.
- Background color: Light lavender (#F3E5F5), offering a soft, inviting backdrop that complements the primary violet.
- Accent color: Electric indigo (#7C4DFF), to highlight calls to action and interactive elements, enhancing the app's dynamic feel.
- Body text and headline font: 'PT Sans', a humanist sans-serif offering both modern appeal and a touch of warmth.
- Use clear and recognizable icons to represent event categories and actions.
- Maintain consistent spacing and alignment throughout the app for a clean and organized look.
- Implement subtle hover effects and transitions for interactive elements to provide visual feedback.
